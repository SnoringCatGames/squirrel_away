# squirrel_away

> **Status: rewrite in progress.** This branch (`master`) is intentionally
> minimal during active development; the real code lives on `dev`.

Squirrel Away — a 2D platformer game using the SnoringCat Godot
frameworks ([scaffolder](https://github.com/SnoringCatGames/scaffolder),
[surfacer](https://github.com/SnoringCatGames/surfacer),
[snore_core](https://github.com/SnoringCatGames/snore_core),
[surf_scaf](https://github.com/SnoringCatGames/surf_scaf)).

Originally written for Godot 3 in GDScript; being rewritten as a Godot
4 GDScript project on top of the now-C++ frameworks.

## Branches

| Branch | Content |
|---|---|
| `master` (this) | Minimal during the rewrite — just this README. |
| `dev` | Active Godot 4 development. |
| `godot3` | Preserved Godot 3 version (pre-rewrite). |
